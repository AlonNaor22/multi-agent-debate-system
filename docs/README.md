# Media assets

Screenshots and the demo GIF referenced from the [root README](../README.md) live here.

**To add the demo:** record a short debate — start one (responses stream in
live) → vote between rounds → final scoreboard — export it as `demo.gif`, drop
it in this folder, then uncomment the image line in the **Demo** section of the
root README. A couple of static screenshots (`setup.png`, `scoreboard.png`) work
just as well if you'd rather not record a clip.
